## 四、关键架构决策记录

| 决策 | 选择 | 理由 |
|---|---|---|
| 嵌入模型 | bge-m3 | 中文支持好，1024 维，多语言 |
| 重排模型 | bge-reranker-v2-m3 | 与嵌入模型同系列，Cross-Encoder |
| 向量数据库 | LanceDB | 本地文件式，无需 Docker |
| BM25 后端 | SQLite FTS5 | 零依赖 |
| 集成方式 | **同进程 import**（而非 HTTP）| 单机单用户场景优 |
| skill 入口模式 | **subprocess 隔离** | 任意 python 都能调，免疫路径问题 |
| 默认 rerank | **关闭**（懒加载）| 默认省 2.3 GB RAM |
| HF_HUB_OFFLINE | **默认开启** | 防止启动时联网 |
| 蒸馏调度 | **兜底机制**（v0.2.4）| Task Scheduler 漏跑时自动补 |
